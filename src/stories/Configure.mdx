# Mayele Design System

Welcome to the **Mayele Design System**, built for the frontend assignment.  
It documents reusable UI components, tokens, and layout patterns created with:

- **Framework**: React + Vite
- **Styling**: SCSS modules with design tokens (`_colors.scss`, `_typography.scss`, `_spacing.scss`)
- **Motion**: Framer Motion for smooth interactive effects
- **Component explorer**: Storybook (autodocs enabled)
- **Accessibility**: a11y addon used to ensure WCAG AA compliance
- **Testing**: Vitest + Storybook accessibility checks

---

## Project Setup

Run locally:

```bash
npm install
npm run storybook
```
